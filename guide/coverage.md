# Coverage

Parity does not execute tests. It reads coverage output generated by your own tools.

## Preferred Format

Use formats with per-line and per-test attribution whenever available. For PHP, PHPUnit XML directories generated with `--coverage-xml` provide the detail needed for matched coverage and attribution checks.

## Portable Format

Clover XML and Cobertura XML work across many ecosystems and support per-file coverage thresholds. They do not include enough detail to prove which test covered which line.

```yaml
coverage_xml: [coverage-xml, clover.xml, cobertura.xml]
```

Parity uses the first configured coverage path that exists.
